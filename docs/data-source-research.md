# 真实数据源探索报告

本报告用于第一阶段真实数据接入 MVP。当前原则是免费、可本地脚本验证、token 不进入前端、缺失字段明确标记，不把 mock 数据伪装成真实数据。

## 结论

- 第一阶段主路径：A 股用 AKShare，A 股历史行情失败时用 BaoStock 兜底，港股用 yfinance 兜底。
- 实施中增加腾讯公开行情作为 A 股行情兜底，原因是 AKShare 东方财富现货端点可能出现远端断开；兜底只用于行情、PE/PB、市值，不替代后续正式数据源评估。
- Tushare Pro 预留 token 配置，但不作为默认主源。
- 巨潮资讯第一阶段只适合公告元数据和 PDF 链接，不做 PDF 正文解析。
- 宏观数据第一阶段先保留接口研究，不进入股票闭环。

## 第一阶段风险

- 免费源的字段和接口可能改变，必须保留 `missing/error/stale`。
- 港股 yfinance 当前本机未安装；若未安装，港股字段会显示缺失，不影响 A 股闭环。
- AKShare 财务指标口径不等同于审计后的完整财报三表；只能作为看板测试字段，不作为投资结论。
- 数据刷新脚本不得并发高频请求，也不得覆盖掉上一次成功缓存为空文件。
